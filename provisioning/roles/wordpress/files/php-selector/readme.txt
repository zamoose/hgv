Use a drop down menu to select different versions of PHP to build the site.

== Description ==

Integration into WP Engine's vagrant development environment.  Use a drop down menu option to trigger the WordPress build
with a different PHP backend processor.

== Installation ==

1. The development environment will install this as a must-use plugin in wp-content/mu-plugins.

== Frequently Asked Questions ==

= I don't see the drop down, where is it? =

The pull down shows up in the WP admin black bar.  Login with your credentials and you should see the drop down in the black bar.
